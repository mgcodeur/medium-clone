## Installation

```bash
npm install
```

## Development server

```bash
npm run watch:css
npm run watch:js
```

## Build for production

```bash
nohup npm run build:css > /dev/null 2>&1 &
nohup npm run build:js > /dev/null 2>&1 &
```
## How to run project

- Just open `index.html` in your browser
